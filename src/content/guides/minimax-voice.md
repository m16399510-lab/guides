:::success

本篇会带你把 MiniMax 语音引擎接到 APP 里。先准备好三个关键信息：**API Key**、**Group ID**、**音色 ID**。

:::

## 第一步：获取发声魔力（注册 MiniMax 账号）

我们的语音功能由强大的 MiniMax 提供技术支持。

:::color5
注册账号：前往➡️ [MiniMax 开放平台](https://platform.minimaxi.com/login)注册。

:::

:::color2
领取福利：完成实名认证后，系统会赠送 15元代金券（够你听他们说很久的话了）。

:::

:::color5
获取双密钥（⚠️最容易出错的一步，请务必仔细）：

:::

点击左侧导航栏的 **【接口密钥】 -> 【创建新的 API Key】**。

(注：账号自带一个“体验中心密钥”，那个没用，不要管它)

![](/assets/minimax-voice/step-01.png)

密钥名称随便填（自己认得就行）。

![](/assets/minimax-voice/step-02.png)

:::danger
🚨 高能预警：创建完成后，当场点击【复制】！立刻把它粘贴到你的手机备忘录里。一旦你关掉这个弹窗，这串密钥就会永久隐藏显示！

:::

![](/assets/minimax-voice/step-03.png)

接着，点击左侧的 **【账户信息】**，把你的 **Group ID** 也复制下来，和密钥存在一起。

![](/assets/minimax-voice/step-04.png)

## 第二步：寻找 TA 的声音（获取音色 ID）

:::color2
方案 A：使用官方免费音色库（机械感稍强）

:::

前往➡️ [MiniMax 音色库](https://www.minimaxi.com/audio/voices)，试听喜欢的声音，点击右侧的“三个点”即可复制【音色 ID】。

![](/assets/minimax-voice/step-05.png)

:::color1
方案 B：电影原声还原（需自行研究克隆/找代捏）

:::

想要还原电影演员的真实声音？你需要提取干声进行克隆。

:::color4
**如果你不会克隆：可以联系管理员进微信群找老师“代捏”。**

价格：加急 5元/条（1小时出），普通 3.5元/条（24小时出）。

（声明：管理员不赚差价，纯粹帮大家对接手艺人）。

:::

:::danger
**需要自备材料：1~5 分钟的纯干声样音**（不能有任何 BGM、环境音或杂音）。

:::

:::color5
🎉 白嫖福利（墙裂推荐）：请加入社牛福利 QQ 群（916210704）！群内已建立【音色 ID 共享文档】，热心的小巫师们可以把捏好的音色ID免费分享在里面，直接复制粘贴即可使用！也可以在群里找人拼单捏声音哦。

:::

## 第三步：为猫头鹰装上引擎（APP 内配置）

拿到密钥、Group ID 和音色 ID 后，回到猫头鹰邮局 APP：

:::color1
**全局配置：** 点击首页右下角 **【⚙️ 齿轮图标】**，在 API 配置界面，将你的 **Group ID 和密钥** 粘贴到对应位置，点击保存。

:::

:::danger
🌍 特别注意：如果你的音色（样音）是英文的，请务必打开【双语模式】！这样 AI 会输出“英文+中文翻译”，但双面镜只会朗读纯正的英文原声。

:::

![](/assets/minimax-voice/step-06.png)

:::color1
**角色配置：** 点开你想听声音的 **【角色资料卡】**，将 **音色 ID** 粘贴进去，并根据喜好调节语速。

:::

![](/assets/minimax-voice/step-07.png)

## 第四步：如何使用语音与视频通话？

**🗣️ 收听语音：** 点击对方回复 **气泡右下角** 的 **【小喇叭】** 图标，即可朗读该条消息。

📹 **视频通话：** 点击输入框左侧的 **【+】号** 拓展菜单，选择 **【视频通话】** 即可发起。

## 魔法部安全使用须知（必看排雷指南）

为了保证双面镜不爆炸，请务必阅读以下注意事项：

:::danger
📱 安卓设备须更新：安卓用户请务必下载安装 **最新版 APK**，否则无法使用视频功能！

👉 [点击此处](https://wwbex.lanzoul.com/is5bQ3k5uczg)下载最新安装包 （提取码：cc4c）

:::

:::color5
🔐 权限放行：使用视频通话前，请确保手机已授予 APP **摄像头和麦克风** 权限。

:::

:::color3
👁️ 视觉模型要求：视频通话需要 AI 能够“看图”。请确保你使用的 API 模型支持视觉功能（如果你发送图片功能可以用，视频也就可以用）。 **逆向渠道（如 [薄甲逆] 系列小克）不支持发送图片，会无法读取视频内容。**

:::

:::color1
🎙️ 关于语音输入：目前游戏内接入了浏览器自带及讯飞桥接的“语音转文字”。但因手机机型太杂，无法保证 100% 兼容，可能会出现只出字不发声，或无法录音的情况（可以自己敲字）。本杰驴正在努力优化中，请多包涵。

:::

:::color2
⏳ 关于延迟：视频通话本质上仍是等待 AI 思考并回复，因此无法像真人微信视频那样做到“绝对实时”。就当做是魔法信号跨国传输的延迟吧！

:::

:::success
💰 灵魂拷问：到底怎么扣费？

**聊天的钱（API）：** 视频通话时，虽然 UI 表现为一句一句发出来，但实际上只调用了一次 API。你平时聊天回复多少钱，视频回复一次就花多少钱，不会重复扣费。

**说话的钱（MiniMax）：** 语音朗读的费用是由 MiniMax 平台收取的（消耗你账号里的 15 元代金券），按字数（Token）计费，与你使用的聊天 API 渠道无关。这是两套独立的计费系统哦。

:::
